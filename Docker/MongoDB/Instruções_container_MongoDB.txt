# Para executar os comandos utilizei o terminal Git Bash(Use o de sua preferência).

1- Entrar na pasta "Docker/MongoDB/" e executar o seguinte comando para criar o container(Exemplo da execução em 'docker_exec.PNG'): 
  docker-compose up -d

2- Execute o comando abaixo e verifique se o conatainer 'mongodb_pythondb' foi criado:
  docker ps

3- Se o passos 1 e 2 rodaram ok, teremos a base configurada.

4- Utilzei o Robot 3T como ferramenta GUI para conferir a estrutura e dados da base.
